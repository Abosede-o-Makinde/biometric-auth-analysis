# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email: **Abomaabidemi27@gmail.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

**Response time:** 72 hours acknowledgement; 14 days disclosure window (coordinated disclosure).

## Scope

This tool analyses biometric system configurations and synthetic scores. It does not:
- Store or transmit real biometric data
- Connect to external biometric systems
- Process authentication credentials

Security reports relating to the GDPR assessment logic (incorrect check outcomes, scoring
errors with compliance implications) are in scope.
